# git reset
`git reset`指令用于撤销变更/回退版本，本质上就是向上移动分支

`git reset`可以加三种参数：

|参数| 工作区 | 暂存区 |
|----------|----------|----------|
| --soft    | keep  | keep |
| --hard   | abort   | abort   |
| --mixed(default)    | keep   | abort  |

参数主要的区别就在于回退版本的时候是否保留原来的工作区和暂存区的内容
